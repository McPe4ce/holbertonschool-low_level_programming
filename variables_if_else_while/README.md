# C - Variables, if, else, while

## Description

This project covers the use of **variables**, **conditional statements** (`if`, `else`), and **loops** (`while`) in C. It also explores the ASCII table, printing alphabets and digits, and combining characters using nested logic.

---

## Learning Objectives

- Declare and use variables of types `int` and `char`
- Use `if` and `if...else` statements for conditional branching
- Use `while` loops for iteration
- Use the ASCII table to print characters and digits
- Print combinations of characters without using complex functions

---

## Files

| File | Description |
|---|---|
| `0-positive_or_negative.c` | Prints whether a random number is positive, negative, or zero |
| `1-last_digit.c` | Prints the last digit of a random number and its classification |
| `2-print_alphabet.c` | Prints the lowercase alphabet using `putchar` |
| `3-print_alphabets.c` | Prints both lowercase and uppercase alphabets |
| `4-print_alphabt.c` | Prints the lowercase alphabet except `e` and `q` |
| `5-print_numbers.c` | Prints single-digit numbers from 0 to 9 |
| `6-print_numberz.c` | Prints single-digit numbers using `putchar` only |
| `7-print_tebahpla.c` | Prints the lowercase alphabet in reverse |
| `8-print_base16.c` | Prints all hexadecimal digits in lowercase |
| `9-print_comb.c` | Prints all possible combinations of single-digit numbers |
