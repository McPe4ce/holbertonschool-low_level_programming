# C - Debugging

## Description

This project focuses on **debugging** C programs by identifying and fixing logical and runtime errors. It covers reading and understanding existing code, tracing through execution flow, and correcting incorrect outputs without rewriting the entire program.

---

## Learning Objectives

- Understand what debugging is and why it matters
- Read and trace through existing C code to identify bugs
- Identify logical errors (incorrect results vs. crashes)
- Fix errors while preserving the original program structure
- Use print statements and code analysis to isolate problems

---

## Files

| File | Description |
|---|---|
| `row_sums.c` | Computes and prints the sum of each row in a 2D array (debugged version) |
| `multiplication_table.c` | Prints a multiplication table (debugged version) |
| `sum_to_n.c` | Computes the sum of all integers from 1 to `n` (debugged version) |

---

## Compilation

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 <file.c> -o <output>
```
