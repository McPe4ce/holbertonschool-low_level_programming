# holbertonschool-low_level_programming

## About

This repository contains low-level programming projects completed as part of the **Holberton School** C programming curriculum. Each project focuses on core C concepts, from the basics of compilation and output to pointers, arrays, strings, and performance analysis.

---

## Table of Contents

| Project | Description |
|---|---|
| [hello_world](./hello_world) | Introduction to the C compilation pipeline and basic output functions |
| [variables_if_else_while](./variables_if_else_while) | Variables, conditionals, and loops in C |
| [functions_nested_loops](./functions_nested_loops) | Function declarations, prototypes, nested loops, and header files |
| [more_functions_nested_loops](./more_functions_nested_loops) | Advanced functions and nested loops for shapes and numeric patterns |
| [pointers_arrays_strings](./pointers_arrays_strings) | Pointers, arrays, string manipulation, and memory operations |
| [argc_argv](./argc_argv) | Command-line arguments using `argc` and `argv` |
| [intro_debugging](./intro_debugging) | Debugging techniques and fixing logical errors in C programs |
| [benchmarking](./benchmarking) | Performance measurement, algorithmic complexity comparison, and energy efficiency analysis |
| [simple-calculator](./simple-calculator) | Interactive CLI calculator with basic arithmetic operations |

---

## Requirements

- **OS:** Ubuntu 20.04 LTS
- **Compiler:** GCC with flags `-Wall -Werror -Wextra -pedantic -std=gnu89`
- **Style:** [Betty coding style](https://github.com/holbertonschool/Betty)
- All files must end with a new line
- No more than 5 functions per file
